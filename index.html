Model Interaction

You can interact with the models using Python via OSC. Here is an example:

from pythonosc import udp_client

# Create OSC client
mrt2 = udp_client.SimpleUDPClient("localhost", 3000)

mrt2.send_message("/model", ["/Users/acaillon/Documents/Magenta/magenta-rt-v2/models/mrt2_small/mrt2_small.mlxfn"])
mrt2.send_message("/buffersize", [16384])
mrt2.send_message("/reset", [])

# prompt slot (0-5), prompt value, prompt weight
mrt2.send_message("/prompt", [0, "dubstep", 1.])
mrt2.send_message("/prompt", [1, "piano", .9])

Available Models

VSCHAOS
Spectrogram VAE

A vintage-flavoured neural audio synthesis package based on unsupervised / (semi-)supervised training of spectral information using variational auto-encoders, optimized for Max/MSP or PureData integration.

Variational Auto-Encoder Spectral Synthesis Max/MSP
GitHub page
DDSP
DSP Hybrid

A library of differentiable versions of common DSP functions (synthesizers, waveshapers, filters), allowing these interpretable elements to be used as part of a deep learning model for audio generation.

Differentiable DSP Audio Synthesis Deep Learning
GitHub page
RAVE
Waveform VAE

A variational autoencoder for fast and high-quality neural audio synthesis, designed for real-time performance and integration into DAWs and Max 8.

Variational Auto-Encoder Real-time Audio Neural Synthesis
GitHub page
AFTER
Latent Diffusion

A diffusion-based generative model for real-time audio features transfer and exploration, blending style/timbre with structural inputs in MaxMSP and Ableton Live.

Latent Diffusion Style Transfer Real-time Audio
GitHub page
MRT2
Latent Discrete

A state-of-the-art open-weights model for real-time music generation, featuring a Python library for JAX/MLX inference and a C++ engine for efficient streaming audio.

Symbolic Music Real-time Generation Streaming
GitHub page
